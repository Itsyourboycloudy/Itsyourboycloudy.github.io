Hello! My name is Jesse Torres

I'm a student in the Games, Interactive Media, and Mobile (GIMM) program at Boise State University, with a strong passion for game development, interactive storytelling, and creative coding. My goal is to design and build experiences that are not only engaging but also meaningful and accessible.

I’m always eager to collaborate on innovative ideas and push the boundaries of interactive media. Thanks for visiting my site — I hope you enjoy exploring my work!

Here are some of my recent projects:

Title: Redesign of Letterboxd (UI/UX)

Redesigned the Letterboxd movie review app with a focus on improving user interface clarity and overall user experience. The updated design features a cleaner layout, simplified navigation, and intuitive visual hierarchy to help users more easily browse, review, and track films.

Game Title: A Tale of Two Towers (Design/Art)

Designed visual assets for an augmented reality (AR) game that unfolds by scanning pages of a physical booklet. Each page triggers a different part of the experience—intro animation, knight gameplay, princess gameplay, and outro sequence. Contributed obstacle designs, background art, and custom animations to build an engaging and story-driven AR world.

Title: The COD Zombies Hub (HTML/CSS/D3/React/Bootstrap/JavaScript)

Developed my first responsive mobile data narrative storytelling website using React, Bootstrap, D3.js, HTML, JavaScript, and CSS. Manually curated and input data to create interactive visualizations that communicate a clear narrative. Focused on user-friendly design and meaningful data presentation across multiple graph types and views.

Game Title: Sea Chanty (Unity/C#)

Developed in just two weeks, Sea Chanty is a 2-player local co-op game created in Unity using C#. One player uses a controller, the other uses keyboard controls, competing in fast-paced naval combat. Players shoot at each other and explosive barrels in deep ocean. I programmed all gameplay systems and mechanics, focusing on smooth controls, responsive combat, and chaotic fun.
